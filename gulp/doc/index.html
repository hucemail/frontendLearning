1. 前言

这一系列的文档主要讲解.NET的一些知识，具体内容请查看文档目录。

课程对应的源码在huccode仓库中。由于后面源码比较长，占用的篇幅会很大，所以，在文档讲解中就只贴出关键性源码了，具体的全部源码都可以在源码仓库中查看。

2. Visual Studio常用快捷键

快捷键 说明
F5 断点运行
F6 重新生成解决方案
F10 逐过程执行
F11 逐行执行
F12 转到定义处
Ctrl+F5 运行
Ctrl+F12 转到实现
Ctrl+A 全选
Ctrl+C 复制
Ctrl+X 剪切、可以剪切掉光标所在的那行
Ctrl+V 粘贴
Ctrl+K+D 格式化代码
Ctrl+K+S 外侧代码，如：try+cache等
Ctrl+J 代码智能提示
Ctrl+Shift+A 添加新项
Shift+F12 查找所以引用
Alt+Shift+F10 实现接口、抽象等
